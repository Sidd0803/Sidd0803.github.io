ClaimsTriage

Vertical AI agent for Medicare inpatient claims review

Automates the first pass on Medicare claims: enriches each claim, runs deterministic anomaly checks, gets an LLM approve / deny / escalate decision, drafts the response letter, and routes it to a human reviewer.

  • LangGraph pipeline (enrich → coverage → anomalies → route → decide → draft); high-severity anomalies bypass the LLM for a deterministic auto-decision.
  • Human-in-the-loop reviewer UI — nothing auto-finalizes; every decision can be overridden, annotated, and resolved.
  • Fixed, reproducible 200-claim eval pool with golden-claim regression tests pinning critical invariants (e.g. an ambiguous auto-decision must escalate, never wrongly deny).

PartSelect AI Assistant

Scoped, tool-using chat agent for appliance-parts support

A support agent for PartSelect refrigerator & dishwasher parts — it answers part lookups, model-compatibility checks, and repair questions by calling deterministic tools over a scraped catalog, and refuses out-of-scope requests.

  • A single Claude tool-use loop handles intent routing, entity extraction, and tool selection in one model call — no hand-rolled router.
  • Compatibility is deterministic (table membership, never inferred): five pure lookup tools over an indexed Postgres catalog, each returning a status.
  • Swapped an 11 MB in-memory compatibility index for a single indexed row read, so memory no longer scales with the catalog; the scraper clears TLS/JA3 bot protection via curl_cffi Chrome impersonation.
